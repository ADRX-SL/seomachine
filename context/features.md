# Scalelist Product Features

## Workflow Rules — Never Mix These Up
Scalelist has two distinct enrichment workflows. Always describe them separately:

**Chrome extension workflow**: Used directly on LinkedIn profiles or LinkedIn Sales Navigator. Reps enrich contacts one by one or in bulk without leaving the browser. No export needed.

**CSV upload workflow**: Used when you already have a list of contacts from another source (CRM export, event list, purchased list, etc.). You upload the CSV to Scalelist and get verified emails and mobile numbers back.

Never write examples that combine both in one sentence like "export from Sales Navigator and upload as CSV" — that is not how it works.
