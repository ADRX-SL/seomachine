## Related Data

The competitor list data is also available in spreadsheet form. See **context/scalelist-competitors-list.md** for the full list of Scalelist competitors with:
- Company name
- Company URL
- Capability (Email Finder, mobile number finder, or both)
- Category (Enricher vs. Waterfall)

## Usage for Blog Posts

- Use **scalelist-competitors-list.md** when comparing or listing software tools
- Reference **competitor-analysis.md** for positioning and Scalelist angles vs. Apollo, ZoomInfo, Lusha, RocketReach, etc.
- For alternative pages ("X alternative"), use both the competitors list and competitor-analysis for accurate tool names, URLs, and differentiation
