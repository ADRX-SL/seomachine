## Cross-Linking Rules

- **Cold email posts** → link to each other + homepage + pricing
- **Cold calling posts** → link to each other + homepage (mobile numbers angle) + cold email posts
- **LinkedIn posts** → link to Chrome extension + CSV upload + cold email/calling post
- **Competitor posts** → link to pricing + homepage + relevant feature page

## Linking Rules
1. Minimum 2 internal links per post
2. No orphan posts — every new post linked from at least one existing post
3. Vary anchor text for the same destination
4. Link naturally — if it doesn't serve the reader, skip it
