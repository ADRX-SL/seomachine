## Usage for Blog Posts

When comparing or listing software tools:
- **Enricher** = Direct enrichment (email/phone lookup)
- **Waterfall** = Aggregates multiple data sources
- Use exact company names and URLs from this list
- Tools with "Email Finder + mobile number finder" support both; "Email Finder" only = email only
- For cold calling content, prioritize tools with mobile number finder capability
